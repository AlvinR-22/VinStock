☁ Sinkronisasi Google Sheets
PANDUAN SETUPketuk untuk buka ▾
1
Buka sheets.google.com → buat spreadsheet baru → salin ID dari URL (bagian antara /d/ dan /edit)
2
Buka script.google.com → buat project baru → paste seluruh isi Code.gs yang disertakan → ganti nilai SHEET_ID dengan ID tadi
3
Di editor Apps Script: klik Run ▶ → initSheets (sekali saja) untuk membuat 4 sheet otomatis
4
Klik Deploy → New deployment → Web app. Set "Execute as: Me" dan "Who has access: Anyone" → Deploy → salin URL-nya
5
Tempel URL di kolom bawah → klik Test Koneksi → jika berhasil klik Simpan. Selesai!
💡 Web ini bisa dibagikan ke siapa saja — cukup kirim link halaman ini. Setiap pengguna mengisi URL GAS yang sama dan data akan tersinkron otomatis.
